Overview

This project is a fully integrated financial model and valuation of Apple Inc. built in Microsoft Excel.

The model includes historical analysis, operating forecasts, three-statement projections, DCF valuation, comparable company analysis, precedent transaction analysis, sensitivity analysis, and model audit checks.
